# 👑 BARON V3 (GOD MODE) — РУКОВОДСТВО ПО ЭКСПЛУАТАЦИИ

Добро пожаловать в **Baron C2 V3**, передовую систему управления (C2), разработанную для абсолютного контроля над развернутыми агентами. Это руководство описывает базовые принципы работы, сборку пейлоадов (билдер) и функционал "God Mode".

## 1. БАЗОВАЯ РАБОТА С ПАНЕЛЬЮ

### Данные для доступа (Admin Credentials)
- **Логин по умолчанию:** `admin` или `baron` (зависит от вашей настройки в `data/config.json`).
- **Пароль по умолчанию:** `admin123` или сгенерированный сервером при первом старте.
- **ВНИМАНИЕ**: Панель снабжена системами Tarpitting и Anti-Bruteforce. После 3-х неудачных попыток входа IP-адрес получает теневой бан или искусственные задержки (Timing Mitigation).

### Выделенный IP-Whitelisting (VPN-Only)
- Ваш текущий IP-адрес **81.25.50.119** (и локальные `127.0.0.1`, `192.168.x.x`, `169.254.83.93`) уже вшит в `server.py` (`self.whitelisted_ips`).
- Любой другой IP, пытающийся открыть `/api/panel/login`, получит немедленный пермач-бан (Access Denied / Honeypot trap).
- Если ваш IP динамический, немедленно обновите массив `self.whitelisted_ips` в `server.py`, иначе вы заблокируете сами себя!

### Главный Экран (Сетка Виджетов)
- **Карта Заражений (Map)**: Показывает примерную геолокацию подключенных агентов.
- **Логи**: Поток системных событий, ошибок и ответов от агентов в реальном времени.
- **Connected Clients**: Таблица активных машин. Кликнув по клиенту **правой кнопкой мыши (ПКМ)**, вы вызовете контекстное меню `God Mode` со всеми функциями.

## 2. СБОРКА АГЕНТА (PAYLOAD BUILDER)

Перейдите во вкладку `Builder` в верхнем или левом меню. Важно понимать, какие галочки за что отвечают:

1. **Server URL**: Адрес вашего C2 (например, `https://baron.cc` или ваш IP). Обязательно с http/https.
2. **Beacon (ms)**: Интервал "пульса" агента. По умолчанию 5000 (5 сек). Если поставить 60000, агент будет опрашивать сервер раз в минуту (лучше для скрытности, но панель реагирует медленнее).
3. **Hidden (Run in Background)**: Скрывает окно консоли при запуске EXE (обязательно для реальной работы).
4. **Persistence (Закрепление)**: Прописывает EXE в автозагрузку реестра (`HKCU\Software\Microsoft\Windows\CurrentVersion\Run`).
5. **Anti-Kill (BSOD Protection)**: Делает процесс критическим (`RtlSetProcessIsCritical`). При попытке убить его через Диспетчер Задач — у жертвы выпадет Синий Экран Смерти. *Использовать с осторожностью!*
6. **Disable Defender**: Отправляет команды PowerShell для отключения антивируса и добавляет папку AppData в исключения.
7. **Anti-Analysis (Stealth)**: Проверяет, не запущена ли программа в песочнице (Sandboxie), виртуальной машине (VirtualBox/VMware), или отладчике (Wireshark, x64dbg). Если обнаружит — самоубивается.
8. **Fake Error**: Показывает поддельное окно ошибки при запуске (например, "Missing MSVCP140.dll"), чтобы жертва подумала, что скачанная "игра/программа" просто сломана.

После нажатия `Build`, сервер скачает C# компилятор (`csc.exe`) в фоне и соберет `.exe` файл. Сборка обычно занимает 2-5 секунд.

## 3. ФУНКЦИОНАЛ КОНТЕКСТНОГО МЕНЮ (ПКМ по Клиенту)

Функции модулей, разбитые по категориям:

### 👁️ Surveillance (Слежка)
* **Screen Stream**: Захват экрана в реальном времени (MJPEG поток).
* **Webcam Stream**: Включение скрытой трансляции с веб-камеры. Светодиод все равно загорится (это аппаратная фича многих камер), будьте аккуратны.
* **Microphone**: Прослушивание помещения через микрофон устройства.
* **Keylogger**: Сбор ВСЕХ нажатий клавиш. Нажмите кнопку, чтобы получить дамп напечатанного текста за сессию.
* **Clipboard Monitor**: Захват текста, скопированного в буфер обмена.

### 💰 Extractors (Кража данных)
* **Steal Browsers (Pass/Cookie)**: Дешифровка базы данных паролей, куки-файлов, кредиток из Chrome, Edge, Edge Chromium, Opera, Brave и др. На движке Chromium и Gecko.
* **Steal Crypto Wallets**: Извлечение `.dat` файлов холодных криптокошельков (Bitcoin Core, Electrum, Monero и др.) с диска.
* **Steal Telegram Session**: Копирование и выгрузка папки `tdata` (если нет Telegram Premium пароля, вы можете зайти без СМС).
* **Steal Discord Token**: Инъекция в `%appdata%/discord` или считывание LevelDB для кражи токенов аутентификации (минует 2FA).

### ⚙️ System Control (Управление)
* **UAC Bypass (Fodhelper)**: Попытка повысить привилегии (запустить cmd.exe как Администратор) без показа желтого/синего окна "Разрешить этому приложению..." (UAC). Использует баг автоэлевации в `fodhelper.exe`.
* **Reverse Proxy (SOCKS5)**: Превращает ПК жертвы в прокси-сервер. Позволяет выходить в интернет через IP-адрес жертвы.
* **Reverse Shell / Terminal**: Открытие полноценной командной строки `cmd.exe`.
* **File Manager**: Управление файлами: загрузка, скачивание, удаление с дисков.

### ☠️ Evasion & Destructive (Разрушение и Обход)
* **Disable Defender (Realtime)**: Отправляет жесткие PowerShell команды для поломки антивируса (побочный эффект: может триггернуть AMSI).
* **Melt & Migrate (Inject)**: Удаляет исходный `.exe` файл жертвы и инжектит свой код в другой легитимный процесс (например, `explorer.exe`). Усложняет удаление вируса.
* **Blue Screen (BSOD)**: Мгновенный вызов `KeBugCheckEx` — компьютер жертвы перезагрузится с синим экраном.
* **Lock Screen**: Блокировка сеанса пользователя (`Win+L`).
* **Kill OS / MBR Wipe** *(В разработке)*: Полное стирание загрузочного сектора, превращает систему в нерабочую машину.
